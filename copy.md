# 끼니록 카피 전체

수정 후 `app.js`에서 해당 문자열을 찾아 교체하면 돼요.
변수가 포함된 카피는 `${변수명}` 형태로 표시했어요.

## 홈 탭 — 인사이트 바

score 기반으로 동작함. 조건이 여러 개 해당될 때 score가 가장 높은 항목이 노출됨.
score는 `기본값 + 카운트 × 가중치` 로 계산되며, 범주 간 우선순위는 기본값으로 보장됨.

### 범주 1 — 불편 신호 (score 500–699, 항상 경고보다 우선)

| 조건 | score 기준 | 카피 |
|------|-----------|------|
| 속쓰림 + 더부룩함 동시 | 680 + (합산) × 20 | `속이 좀 불편했네요, 어떤 메뉴였는지 메모에 남겨봐요` |
| 속쓰림만 | 640 + N × 20 | `속쓰림이 있었어요, 자극적인 음식이 있었는지 살펴봐요` |
| 더부룩함 + 졸림 | 620 + N × 20 | `더부룩하고 졸리기도 했네요, 어떤 메뉴였는지 메모에 남겨봐요` |
| 더부룩함 2회 이상 | 600 + N × 20 | `더부룩함이 여러 번이었어요, 어떤 끼니 후였는지 살펴봐요` |
| 더부룩함 1회 | 560 | `더부룩함이 한 번 있었어요, 어떤 메뉴였는지 살펴봐요` |
| 졸림 + 탄수 많음 | 540 + N × 10 | `식후 졸림이 있었어요, 탄수화물 양이 영향을 줬을 수 있어요` |
| 졸림만 | 510 | `식후에 졸리기도 했네요, 식사 속도나 양이 어땠는지 살펴봐요` |

### 범주 2 — 행동 경고 (score 230–430)

| 조건 | score 기준 | 카피 |
|------|-----------|------|
| 배 터질 것 같음 + 빠른 식사 | 430 + 복합 | `빨리 먹고 많이 먹은 끼니가 있었어요, 천천히 먹으면 포만감이 더 잘 느껴져요` |
| 배 터질 것 같음만 | 400 + N × 20 | `많이 먹은 끼니가 있었어요, 다음엔 한 박자 느리게요` |
| 배달 = 오늘 전체 끼니 수 | 370 + N × 20 | `오늘 끼니를 모두 배달로 했어요, 내일은 간단하게 직접 챙겨봐요` |
| 배달 2회 이상 | 350 + N × 20 | `배달을 자주 했어요, 다음 끼니는 집밥 어때요?` |
| 나트륨 2회 이상 | 330 + N × 20 | `짠 게 좀 많았네요, 물을 조금 더 마셔봐요` |
| 탄수 많음 2회+ + 채소·단백질 없음 | 310 + N × 20 | `탄수화물이 많은 편이었어요, 채소나 단백질을 곁들이면 균형이 잡혀요` |
| 탄수 많음 2회 이상 | 290 + N × 20 | `탄수화물이 좀 많았어요, 다음엔 채소나 단백질을 곁들여봐요` |
| 빠른 식사 1회 + 탄수 많음 | 270 + 복합 | `빠르게 먹고 탄수화물도 많았어요, 천천히 먹으면 포만감이 더 잘 느껴져요` |
| 포만감 높음 2회 이상 | 260 + N × 15 | `포만감이 높은 끼니가 여러 번이에요, 다음엔 한 단계 가볍게요` |
| 빠른 식사 2회 이상 | 250 + N × 15 | `빨리 먹은 끼니가 여러 번이에요, 다음엔 조금 천천히요` |
| 식사량 부족 + 채소·단백질 없음 | 240 | `식사량이 부족했을 수 있어요, 채소나 단백질을 더 챙겨봐요` |
| 당 있음 + 집밥 없음 | 230 | `당이 조금 있었어요, 다음 끼니는 담백하게 골라봐요` |

### 범주 3 — 긍정 신호 (score 80–200)

| 조건 | score 기준 | 카피 |
|------|-----------|------|
| 천천히 + 적당 포만감 + 채소 + 단백질 | 200 + 복합 | `균형 있게 잘 먹었어요` |
| 천천히 + 적당 포만감 | 185 + 복합 | `천천히, 적당히 먹었어요` |
| 천천히 먹음만 | 170 | `천천히 먹은 끼니가 있었어요, 좋은 습관이에요` |
| 채소 + 단백질 + 탄수 없거나 적음 | 160 + 복합 | `채소, 단백질, 탄수까지 균형 있게 챙겼어요` |
| 채소 + 단백질 | 150 + 복합 | `채소와 단백질을 모두 챙겼어요` |
| 속편함 + 적당 포만감 | 140 + 복합 | `속도 편하고 포만감도 좋았어요, 오늘 식사 패턴을 기억해둬요` |
| 속편함만 | 130 + N × 15 | `속이 편한 하루였네요` |
| 채소 또는 단백질 | 120 + 복합 | `채소나 단백질을 챙겼어요` |
| 적당 포만감 2회 이상 | 110 + N × 15 | `포만감을 잘 조절한 하루예요` |
| 적당 포만감 1회 | 100 + N × 15 | `적당한 포만감으로 먹었어요` |
| 가볍게 먹음 (식사량 부족 없음) | 90 | `산뜻하게 먹었어요` |
| 가볍게 먹음 (식사량 부족 있음) | 80 | `산뜻하게 먹었어요, 든든하기도 했는지 살펴봐요` |

## 요약 탭 — 이번 주 패턴 인사이트

score 기반. 조건이 여러 개 해당될 때 score가 가장 높은 항목이 노출됨.
`${태그}이/가` 조사는 `josa()` 함수로 자동 처리됨.

### 연속 기록 마일스톤 (score 800+)

| 조건 | score | 카피 |
|------|-------|------|

| 연속 14일 이상 | 820 + N | `${N}일 연속 기록이에요, 이 정도면 패턴이 꽤 보일 거예요` |

| 속쓰림 2회 | 640 + N × 20 | `속쓰림이 두 번 있었어요, 어떤 메뉴 후에 나타나는지 살펴봐요` |
| 더부룩함 3회+ + 빠른 식사 3회+ | 620 + 복합 | `더부룩함과 빠른 식사가 겹치고 있어요, 천천히 먹으면 속이 나아질 수 있어요` |
| 더부룩함 3회 이상 | 600 + N × 15 | `더부룩함이 이번 주 ${N}번이에요, 어떤 끼니 후에 나타나는지 살펴봐요` |
