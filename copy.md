# 끼니록 카피 전체

수정 후 `app.js`에서 해당 문자열을 찾아 교체하면 돼요.
변수가 포함된 카피는 `${변수명}` 형태로 표시했어요.

---

## 홈 탭 — 날짜 네비

| 위치 | 현재 |
|------|------|
| 날짜 버튼 서브라벨 (오늘) | `오늘` |
| 날짜 버튼 서브라벨 (과거) | `지난 기록` |
| 이전 날짜 버튼 aria-label | `이전 날짜` |
| 다음 날짜 버튼 aria-label | `다음 날짜` |
| 날짜 피커 "오늘로" 버튼 | `오늘로 이동` |

---

## 홈 탭 — 인사이트 바

조건 우선순위 순서대로. 변수 포함 카피는 `${N}` 형태.

| 조건 | 현재 카피 |
|------|-----------|
| 기록 없음 (오늘) | `오늘 첫 끼니를 남겨봐요` |
| 더부룩함 + 졸림 | `더부룩하고 졸리기도 했네요, 어떤 메뉴였는지 메모에 남겨봐요` |
| 더부룩함만 | `더부룩함이 있었어요, 어떤 메뉴였는지 살펴봐요` |
| 졸림 + 탄수 많음 | `식후 졸림이 있었어요, 탄수화물 양이 영향을 줬을 수 있어요` |
| 배달 2회 이상 | `배달을 자주 했어요, 다음 끼니는 집밥 어때요?` |
| 나트륨 2회 이상 | `짠 게 좀 많았네요, 물을 조금 더 마셔봐요` |
| 탄수 많음 2회 이상 | `탄수화물이 좀 많았어요, 다음엔 채소나 단백질을 곁들여봐요` |
| 배 터질 것 같음 있음 | `많이 먹은 끼니가 있었어요, 다음엔 한 박자 느리게요` |
| 포만감 높음 2회 이상 | `포만감이 높은 끼니가 여러 번이에요, 다음엔 한 단계 가볍게요` |
| 빠르게 먹음 2회 이상 | `빨리 먹은 끼니가 여러 번이에요, 다음엔 조금 천천히요` |
| 식사량 부족 + 채소/단백질 없음 | `식사량이 부족했을 수 있어요, 채소나 단백질을 더 챙겨봐요` |
| 당 있음 + 집밥 없음 | `당이 조금 있었어요, 다음 끼니는 담백하게 골라봐요` |
| 천천히 먹음 + 적당 포만감 | `천천히, 적당히 먹었어요` |
| 천천히 먹음만 | `천천히 먹은 끼니가 있었어요, 좋은 습관이에요` |
| 채소 + 단백질 둘 다 | `채소와 단백질을 모두 챙겼어요` |
| 채소 또는 단백질 | `채소나 단백질을 챙겼어요` |
| 적당 포만감 2회 이상 | `포만감을 잘 조절한 하루예요` |
| 적당 포만감 1회 | `적당한 포만감으로 먹었어요` |
| 가볍게 먹음 | `산뜻하게 먹었어요, 든든하기도 했는지 살펴봐요` |
| 기본 (해당 없음) | `몸 상태도 메모에 남겨봐요` |

## 요약 탭 — 이번 주 패턴 인사이트

| 조건 | 현재 카피 |
|------|-----------|
| 기록 없음 + 지난 달 기록 있음 | `지난달에 ${N}번 기록했어요, 이번 주도 시작해봐요` |
| 기록 없음 (처음) | `첫 끼니를 기록하면 이번 주 패턴을 같이 살펴볼게요` |
| 연속 14일 이상 | `${N}일 연속 기록이에요, 이 정도면 식습관 패턴이 꽤 보일 거예요` |
| 더부룩함 3회 이상 | `더부룩함이 이번 주 ${N}번이에요, 어떤 끼니 후에 나타나는지 살펴봐요` |
| 배달 4회 이상 | `이번 주 배달이 ${N}번이에요, 집밥이나 간단한 요리로 한두 번 바꿔볼까요` |
| 나트륨 3회 이상 | `짠 음식이 이번 주 ${N}번이에요, 나트륨이 쌓이면 부기로 나타날 수 있어요` |
| 빠른 식사 4회 이상 | `이번 주 ${N}끼를 빠르게 먹었어요, 천천히 먹으면 포만감이 더 잘 느껴져요` |
| 야식 3회 이상 | `야식이 이번 주 ${N}번이에요, 저녁 식사 시간을 조금 앞당겨 볼까요` |
| 채소 4회+ AND 단백질 3회+ | `채소와 단백질을 꾸준히 챙겼어요, 이번 주 균형이 좋아요` |
| 채소 4회 이상 | `채소를 이번 주 ${N}번 챙겼어요, 단백질도 같이 챙기면 더 좋아요` |
| 적당 포만감 5회 이상 | `포만감이 적당했던 끼니가 ${N}번이에요, 식사 조절이 잘 되고 있어요` |
| 채소 0회 + 5끼 이상 기록 | `이번 주 채소 기록이 없어요, 다음 주엔 한 끼라도 채소를 곁들여봐요` |
| watch 태그 2회 이상 (최다) | `${태그}이 이번 주에 ${N}번 나왔어요, 다음 주엔 조금 줄여봐요` |
| 한 달 20회 이상 기록 | `최근 한 달 ${N}번 기록했어요, 패턴이 잘 보이고 있어요` |
| 기본 (해당 없음) | `이번 주 ${N}끼 기록했어요, 꾸준히 쌓아가고 있어요` |

인사이트 카드 h3 (기록 있을 때) | `이번 주 기록이에요`
인사이트 카드 h3 (기록 없을 때) | `기록을 시작해봐요`
인사이트 카드 배지 | `이번 주 패턴`
