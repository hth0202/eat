# 끼니록 UI/UX 라이팅 사전

## 브랜드 톤

끼니록은 식단을 평가하는 앱이 아니라, 사용자가 자기 식습관을 가볍게 살펴보는 앱이다.

- 판단보다 관찰
- 관리보다 살핌
- 실패보다 다음 선택
- 정확한 분석보다 패턴
- 많은 입력보다 빠른 기록

## 태그 라벨 원칙

태그는 짧고, 사용자가 바로 이해하는 말로 쓴다. 전문 영양 용어보다 생활 언어를 우선한다.

| 피하기 | 쓰기 |
| --- | --- |
| 단순당 | 당(설탕) |
| 당류 음료 | 액상과당 |
| 고나트륨 | 나트륨(염분) |
| 자극적임 | 매움 |
| 튀김/기름짐 | 기름짐 |
| 외부 음식 | 배달/외식 |
| 소화 불편 | 더부룩함 |

## 태그 카테고리

| 카테고리 | 용도 | 예시 |
| --- | --- | --- |
| 조금 줄여볼 것 | 많이 보이면 다음 끼니에서 살짝 조절해요 | 밀가루, 당(설탕), 액상과당, 카페인, 기름짐, 매움, 나트륨(염분) |
| 더 챙길 것 | 자주 보이면 좋은 신호예요 | 채소, 단백질, 물, 집밥, 과일 |
| 먹은 방식 | 배달, 외식, 야식처럼 식사 상황을 남겨요 | 배달/외식, 야식 |
| 먹고 난 상태 | 속이 편했는지, 졸렸는지도 같이 볼 수 있어요 | 속 편함, 졸림, 더부룩함 |

| --- | --- |

## 상태 문구

| 상황 | 문구 |
| --- | --- |
| 기록 없음 | 아직 기록이 없어요. 가볍게 기록해 볼까요? |
| 오늘 첫 기록 전 | 완벽하게 적지 않아도 괜찮아요. 하루 한 끼만 남겨 보세요. |
| 기록 시작 | 기록이 쌓이기 시작했어요. 오늘 몸이 어땠는지도 메모해 두면 도움이 돼요. |
| 좋은 패턴 | 건강에 좋은 기록이 쌓였어요. 이런 작은 기록이 건강한 식습관을 만들어요. |
| 배달/외식 반복 | 오늘은 배달/외식을 자주 했어요. 다음 끼니는 집밥이나 간단한 조합으로 맞춰볼까요? |
| 나트륨 반복 | 오늘은 조금 짜게 먹었어요. 물이나 담백한 메뉴로 균형을 맞춰봐도 좋아요. |
| 더부룩함 | 먹고 난 뒤 더부룩함이 남아 있어요. 어떤 메뉴에서 그랬는지 메모해두면 다음 선택에 도움이 돼요. |
| 탄수화물 많음 | 오늘은 탄수화물 섭취가 조금 많았어요. 다음 끼니에서는 채소나 단백질을 곁들여봐도 좋아요. |
| 당/액상과당 | 오늘은 당이나 액상과당이 조금 많았어요. 다음 끼니에서는 물이나 담백한 메뉴를 골라봐요. |
| 빠른 식사 | 10분 안에 먹은 끼니가 여러 번 있었어요. 다음 끼니는 몇 입만 천천히 먹어봐도 좋아요. |
| 천천히 먹음 | 천천히 먹은 끼니가 있었어요. 여유 있게 먹는 습관은 오래 가져가도 좋아요. |

## 수정 판단 기준

새 문구나 UI를 바꿀 때는 아래 셋 중 하나에 해당할 때만 바꾼다.

1. 기록 시간이 줄어드는가
2. 식습관 판단이 더 정확해지는가
3. 헷갈리는 선택지가 줄어드는가

해당하지 않으면 다음 테스트 주기까지 보류한다.
