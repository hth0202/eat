# 끼니록 UI 디자인 시스템

## 레이아웃 원칙

- 모바일 기준 너비는 460px 안에서 잡는다.
- 화면 기본 좌우 여백은 16px이다.
- 주요 섹션 간격은 32px, 카드 내부 여백은 16px을 기본으로 한다.
- 기록 작성 화면은 바텀시트가 아니라 전체 작성 화면으로 쓴다.
- 작성 화면에서는 헤더와 저장 영역을 고정하고, 입력 영역만 스크롤한다.

## 간격 토큰

| 토큰 | 값 | 용도 |
| --- | --- | --- |
| `--space-1` | 4px | 미세 간격 |
| `--space-2` | 8px | 칩 내부/작은 간격 |
| `--space-3` | 12px | 요소 사이 기본 간격 |
| `--space-4` | 16px | 화면 좌우/카드 기본 여백 |
| `--space-5` | 20px | 필드 간격 |
| `--space-6` | 24px | 헤더/상단 간격 |
| `--space-7` | 32px | 섹션 간격 |

## 반경 토큰

| 토큰 | 값 | 용도 |
| --- | --- | --- |
| `--radius-sm` | 12px | 작은 버튼 |
| `--radius-md` | 16px | 입력창, 탭 |
| `--radius-lg` | 20px | 기본 카드 |
| `--radius-xl` | 24px | 강조 카드 |

## 글자 위계

- 앱명: 24px / 750
- 화면 제목: 19px / 750
- 카드 제목: 16-17px / 700
- 버튼, 칩: 13px / 600
- 설명문: 12-14px / 400-500
- 숫자: 26px / 700

## 기록 카드

- 오늘 화면에는 기록된 끼니만 보여준다.
- 기록이 없을 때는 CTA 카드 하나만 보여준다.
- 식단 추가는 하단 내비게이션 중앙의 플러스 버튼에서 시작한다.

## 작성 화면

- 전체 화면으로 연다.
- 상단: 끼니명, 닫기 버튼 고정
- 본문: 입력 필드만 스크롤
- 하단: 삭제/닫기/저장 액션 고정
